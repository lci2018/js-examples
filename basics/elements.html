JS Elements

Data

Data is the information that use the scripts to work. The usual types are numerals (a numeric value), literals (a textual value, usually in quotation marks) and booleans (with a true or false value).

Variables

Variables are keywords which value can change. They can reference data thay may be unknown at a given moment, but will be interpreted and manipulated by our scripts.

Functions

Functions group several lines of code that can be executed at our will. They are declared with the word function followed by the function name and a pair of brackets, and the related group of code in curly brackets.

Conditionals

Conditionals execute a group of code only when certain conditions are met (for example, when a variable equals a specific value).

Loops

Loops repeat the execution of a group of code a specific number of times (for example, for each element of a given type, or when a specific condition is met).

Objects

Objects are data models with properties (relation between a key and its value) and methods (internal functions). Almost everything in JavaScript are objects, like the window objects, the console object and even variables and functions and, as such, these objects have methods and properties that can be accessed.

Accessing the DOM

The DOM is the Document Object Model (how JavaScript interprets a website). Methods from this object, like document.querySelector(), are used to access and manipulate the HTML.

Events and listeners

Events are things that happen on a website, originated either by a user action (clicking, scrolling, pressing a key…) or by the browser itself (starting or finishing to load a page, exiting a page…).

Listeners are methods that link a specific function with a specific event; for instance, when the action of clicking a button occurs, a function that shows some content is executed.
